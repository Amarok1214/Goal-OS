<objective>
Add focus productivity features: Focus History tracking, Pomodoro Timer, Goal Categories, and enhanced Progress tracking.
</objective>

<execution_context>

# Focus History (focusStore.ts)
- Track sessions: id, taskId, taskTitle, startTime, endTime, duration, mode
- getTodayFocusTime(): sum of today's session durations
- clearHistory(): wipe all sessions
- Sessions persist to localStorage

# Pomodoro Timer (FocusTimer.tsx)
- Two modes: "Focus" (freeform) and "Pomodoro" (25/5 cycles)
- Pomodoro phases: work, break, idle
- Auto-switch between work and break
- Browser notifications when phase completes
- Progress bar within Pomodoro mode

# Categories (types/index.ts, GoalForm.tsx, GoalCard.tsx)
- GoalCategory: work, health, learning, finance, creative, social, other
- CATEGORY_LABELS: display names with emoji
- CATEGORY_COLORS: color scheme for each category
- Category selector in GoalForm
- Category badge on GoalCard

# Enhanced Progress (GoalCard.tsx)
- Count: tasks + subtasks = total items
- Completed: task completions + subtask completions
- Display: "X/Y items (A/B tasks + C/D subtasks)"
</execution_context>
