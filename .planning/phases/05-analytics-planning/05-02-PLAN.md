## Objective

Create a Weekly Study Planner that allows students to schedule study blocks for the week, creating structure and reducing decision fatigue. Blocks can optionally link to existing goals.

## Context

Students often struggle with "what should I study now?" The planner solves this by letting them pre-schedule study sessions. When a scheduled block is active, the Pomodoro timer highlights it, providing context and motivation.

## Success Criteria

- [ ] Weekly grid view displays 7 days with time slots (6 AM - 11 PM)
- [ ] User can add time blocks to any day/time
- [ ] Blocks persist in localStorage across refreshes
- [ ] Blocks can optionally link to existing goals
- [ ] Week navigation (prev/next week)
- [ ] Blocks display goal title if linked
- [ ] Dark theme consistent with app design

### Block Positioning Math
```typescript
const getBlockStyle = (block: ScheduleBlock) => {
  const hour = parseInt(block.startTime.split(':')[0])
  const minute = parseInt(block.startTime.split(':')[1])
  const duration = getDurationMinutes(block.startTime, block.endTime)
  
  const top = ((hour - 6) * 60 + minute) * (rowHeight / 60)
  const height = duration * (rowHeight / 60)
  
  return { top: `${top}px`, height: `${height}px` }
}
```

## Data Model

### ScheduleBlock
```typescript
interface ScheduleBlock {
  id: string
  dayOfWeek: 0 | 1 | 2 | 3 | 4 | 5 | 6  // 0=Sunday
  startTime: string   // "09:00" (24hr format)
  endTime: string     // "10:30"
  goalId?: string     // Optional link to goal
  title: string      // "Math Study" or auto-filled from goal
  subject?: string    // For future categorization
  recurring: boolean  // Repeats weekly
  color?: string      // Hex color override
  createdAt: Date
}
```

### Store Actions
```typescript
interface ScheduleStore {
  blocks: ScheduleBlock[]
  
  addBlock: (block: Omit<ScheduleBlock, 'id' | 'createdAt'>) => string
  updateBlock: (id: string, updates: Partial<ScheduleBlock>) => void
  deleteBlock: (id: string) => void
  
  // Queries
  getBlocksByDay: (day: number) => ScheduleBlock[]
  getBlocksForWeek: (weekStart: Date) => ScheduleBlock[]
  getCurrentBlock: () => ScheduleBlock | null
  
  // Computed
  getTotalPlannedMinutes: (weekStart: Date) => number
}
```
