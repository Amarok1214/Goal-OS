## Objective

Create a Study Dashboard that displays focus metrics, productivity trends, and study insights using existing Pomodoro session data. Helps students understand their study patterns.

## Context

The app already tracks focus sessions in `focusStore.sessions[]` and distraction logs in `focusStore.distractionLogs[]`. This plan leverages that data to create a visual analytics view without requiring new data collection.

## Success Criteria

- [ ] Dashboard view accessible from sidebar navigation
- [ ] Stats cards display: Total hours, Sessions count, Goals completed, Focus streak
- [ ] Bar chart shows daily focus time for the current week
- [ ] Data updates when sessions are completed
- [ ] Dark theme consistent with app design

### Key Calculations
```typescript
// Total hours this week
const getWeekMinutes = () => {
  const weekStart = getStartOfWeek(new Date())
  return sessions
    .filter(s => s.startTime >= weekStart && s.type === 'work')
    .reduce((acc, s) => acc + s.duration, 0) / 60
}

// Focus streak
const getStreak = () => {
  let days = 0
  let date = new Date()
  while (getDailyMinutes(date) > 0) {
    days++
    date.setDate(date.getDate() - 1)
  }
  return days
}
```
